
This document contains a list of files of the data and their descriptions.

Crease Pattern Diagrams (in .svg format) are created with Sketchup and Inkscape.
Origami Simulator can be accessed at https://origamisimulator.org/.

Files	------------------------   Descriptions

    DataDescription-Bloom_Pattern_Data.txt	This document
    README.txt					Introduction & brief summary of all contents
    Yoshimura Bloom Pattern Computer Program/		A computer program written in Python and executed with Python Launcher (3.9 or above) that generates Yoshimura bloom patterns of the form RH-Y-m.h(1/2)
        Yoshimura_bloom_pattern_code/	
	    README.txt		A user manual on how to use this program.
            execute.py		run this file with Python Launcher to execute the program
            Bloom_Yoshimura.py	main body of the program (NOTE: you can save the output as a .svg file by editing a line of code at the end of the file)
            settings.py		a document to enter your preferences.
            examples/		Example outputs of this computer program.
                Y6-2 inverted.png
                Y6-2 panels.png
                Y18-10.png
                Y6-2 monochromatic.png
                Y6-2 points.png
